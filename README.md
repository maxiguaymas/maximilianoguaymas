## 👨‍💻 ¿Quién soy?

Soy un apasionado de la tecnología con una formación sólida como **Analista de Sistemas**. Mi enfoque no se detiene en la escritura de código; mi valor agregado reside en la capacidad de comprender el ecosistema de un negocio, relevar requerimientos críticos y diseñar arquitecturas que sean escalables y útiles para el usuario final.

Creo firmemente que el software debe ser una herramienta que simplifique procesos, no que los complique.

---

## 🎯 ¿Por qué este portafolio?

Este espacio nace de la necesidad de materializar mi visión sobre el desarrollo web moderno. Decidí construirlo desde cero para demostrar:

- **Dominio Técnico:** Implementando las últimas versiones de **Next.js 15** y **Tailwind CSS v4**.
- **Arquitectura Segura:** Uso de *Server Actions* para el manejo de datos sensibles (como el formulario de contacto) sin exponer lógica al cliente.
- **Experiencia de Usuario (UX):** Una interfaz minimalista, fluida y con animaciones sutiles que guían la atención del reclutador o cliente.
- **Compromiso con la Calidad:** Un código limpio, tipado con TypeScript y optimizado para SEO y rendimiento.
